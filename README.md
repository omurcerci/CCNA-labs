# CCNA-labs
📌 Overview
This project is a simulation of a simple home environment LAN setup and access to the target area via CPT.
🎯 Objectives
Connecting a PC, smartphone, and laptop to a home router and then establishing a network connection from the home router to a target server is a crucial step in the process.
🖧 Topology
1 PC
1 Laptop
1 Smartphone
1 HomeRouter
